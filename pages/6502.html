6502 Emulator

The goal of this program is for it to become a tool to help me write and test programs for my 6502 based computer. Right now it only performs the functionality of a basic 6502 and in the future I will have to add the extended operations/addressing modes included in the actual 6502 model (WD65C02) that I have.

I am also going to include in this program an emulator for the TMS9918 that I have for graphics output. That is a much bigger next step in the project.
