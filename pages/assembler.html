Assembler

This is a program I wrote in C that takes a Hack Assembly Language file as input and outputs a binary file for the Hack CPU, the language and CPU both being outlined in the book "The Elements of Computing". This project changed the way I looked at programming languages. Before I thought they had some sort of special properties, but now I know programming files are no different then text files. The "language" is defined by the compiler program.
