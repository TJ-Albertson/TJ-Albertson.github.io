3D Model Viewer

Learnopengl.com was a huge resource in helping me make this. Although I can safely say that I've touched, if not entirely rewritten, every line of code in this project. All of the code on Learnopengl.com is Object Oriented which I am not a fan of.

The guest article Skeletal Animation on learnopengl.com was also very helpful. I believe I was able to improve on the animation part, although I haven't tested it yet, by adding a item to the Node struct to store an important part of the transformation. In the article these transformations were stored in a map that was looked up for every transformation for every bone every frame.
