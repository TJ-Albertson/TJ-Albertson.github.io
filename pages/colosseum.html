Colosseum

This is a movie rating website built with Ruby on Rails and an SQL database. I use a basic authentication system, just hashing the password and storing it in the database. The site and database are hosted on Heroku.

A user can search movies, either give the movie a thumbs up or down, and view other users profiles to see what they thought of the movie.

I used the Movie Database API to get the movie data and thumbnails.
